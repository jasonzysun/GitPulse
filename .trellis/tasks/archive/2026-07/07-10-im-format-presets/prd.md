# 飞书/钉钉/企微格式适配

## Goal

内置常见企业通讯工具的周报格式预设模板，让用户一键复制成目标平台友好的格式，消除"写完报告还要手动调格式"的痛点。

## Requirements

### 预设模板

- **飞书文档格式**：支持飞书 Markdown 子集（加粗、列表、分隔线），避免飞书不支持的语法
- **钉钉日志格式**：适配钉钉日志的纯文本结构（标题行 + 缩进列表）
- **企业微信格式**：适配企微消息的 Markdown 格式（支持有限的 Markdown 语法）
- **Confluence 格式**：适配 Confluence wiki 语法
- **纯文本格式**：去除所有 Markdown 标记，适合粘贴到任何地方

### 实现方式

- 复用现有的报告格式模板系统（`{periodLabel}`, `{commitItems}` 等变量）
- 在设置 → 报告格式中新增"预设模板"下拉，选择后自动填充对应模板
- 用户可在预设基础上二次编辑
- 预设模板切换不影响已生成的历史报告

### 前端

- 报告预览区域增加"复制为..."按钮或下拉菜单，可选择目标格式
- 复制时实时转换格式，不影响存储的原始报告

## Acceptance Criteria

- [ ] 至少 5 种预设模板（飞书、钉钉、企微、Confluence、纯文本）
- [ ] 预设模板选择后正确填充到模板编辑器
- [ ] 用户可在预设基础上修改
- [ ] "复制为..."功能可实时转换格式
- [ ] 各平台格式在对应平台中粘贴后排版正确
